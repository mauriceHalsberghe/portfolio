An API fetching recipe website

Recept Web

Recept Web is a recipe website that fetches data from an API, offering a wide range of delicious recipes. With a user-friendly design and real-time updates, it makes discovering and exploring new dishes easy and enjoyable.

Recept Web is an interactive recipe website that pulls data from an external API to display a wide array of delicious dishes. Users can browse, create, edit, and delete recipes directly within the interface, making it a dynamic cooking companion. The site features a built-in dark mode toggle, offering a modern user experience that adjusts to user preferences. As my first project working with APIs, it helped me understand how to fetch, display, and manipulate data asynchronously. With a responsive design and real-time updates, Recept Web combines functionality and aesthetics for food lovers of all kinds.
